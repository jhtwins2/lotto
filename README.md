# 🎱 로또 번호 생성기

> 무료 · 무제한 · 광고 없음 · 동행복권 공식 색상

동행복권 공식 색상 규격에 맞춰 제작한 로또 번호 생성기 웹 앱입니다.
1~45 숫자를 탭해서 제외할 번호를 선택하고, 한 번에 5줄을 생성합니다.

---

## ✨ 주요 기능

- 🎯 **5줄 동시 생성** — 로또 1장(5게임) 분량을 한 번에 뽑아줌
- 🔴 **터치로 제외 숫자 선택** — 1~45 버튼을 탭하면 빨간색으로 제외 표시
- 🎨 **공식 색상 적용** — 동행복권 공식 번호별 공 색상 규격 그대로
- 🛡️ **안전장치** — 제외 숫자 최대 39개 제한 (6개는 반드시 남김)
- 📱 **PWA 지원** — 홈 화면에 추가하면 앱처럼 사용 가능

---

## 🎨 공식 색상 규격 (동행복권)

| 번호 범위 | 색상 |
|----------|------|
| 1 ~ 10 | 🟡 노랑 |
| 11 ~ 20 | 🔵 파랑 |
| 21 ~ 30 | 🔴 빨강 |
| 31 ~ 40 | ⚫ 회색 |
| 41 ~ 45 | 🟢 초록 |

---

## 📱 사용 방법

### 설치 (홈 화면 추가)
1. 스마트폰 브라우저로 접속
2. 메뉴 → **홈 화면에 추가**
3. 앱 아이콘으로 바로 실행

### 번호 생성 순서
1. 제외하고 싶은 숫자 탭 (빨간색으로 표시)
2. 다시 탭하면 제외 해제
3. **번호 생성** 버튼 클릭
4. A~E 5줄 번호 확인
5. **↺** 버튼으로 전체 초기화
